# SDET_challenge JSON Schema Lint

Heroku-deployable version of an online JSON schema linter for validating responses to an [SDET challenge](https://github.com/aliking/sdet_challenge). Designed so that technical screeners can get an idea of how well candidates have done at a take-home coding challenge. 

# Local
## Setup
```
npm install
cd draft4/
bower install
```
## Preview

```
PORT=3456 node server.js
```
Open http://localhost:3456

## Building

```
grunt
```
